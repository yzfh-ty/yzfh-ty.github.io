一、准备

1.1 APP

  • NFC Tools PRO_8.6.1.apk （点击下载） 备用：点击下载
  • NFC Tasks_5.4.4.apk （点击下载） 备用：点击下载

1.2 材料

（1） NFC标签

  • NTAG213容量一般是144字节

  • NTAG215容量一般是504字节(我建议买这个，第一个容量稍微有点小）

  • NTAB216容量一般是888字节

详细参数参考: NTAG213 215 216.pdf （点击下载） 备用：点击下载

（2） 可以用NFC的手机（NFC基本都在摄像头有旁边）

二、教程

1 制作

（1） 打开下载好的NFC Tools PRO

（2） 点击任务

（3） 点击添加任务

（4） 选择应用程序

（5） 再次选择应用程序

（6） 选择程序包名（也可以手动输入）

（7） 这里可以搜索,点击网易云音乐

（8） 点击OK

（9） 再次点击添加任务

（10） 选择应用程序

（11） 点击URL/URI

（12） 输入orpheus://song/<歌曲id>/?autoplay=1（仅适用于网易云）

tips：其他音乐APP参考快捷指令url大全

（13） 点击OK

（13） 点击写入

注意：这时候不要将NFC标签贴近手机NFC区域

（14） 将NFC标签贴近手机NFC区域

（15） 显示这个就已经说明成功了，可以愉快的玩耍了

2 网易云id在哪

（1） 电脑版

点击分享，选择复制链接

这个链接中标出部分就是

（2） 手机版

  • 打开正在播放的歌曲

  • 在浏览器打开，然后点击上面搜索栏

  • 点击复制

  • 图中标出的部分就是歌曲id
